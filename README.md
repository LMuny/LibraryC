# LibraryC
Library containing the fonction for c

1 - ft_strlen  : Returns the lenght of a string in an int

2 - ft_isalpha : Returns "1" if the char is a letter

3 - ft_isdigit : Returns "1" if the char is a number

4 - ft_isalnum : Returns "1" if the char is a letter or a number

5 - ft_isascii : Returns "1" if the char is in the ASCII table

6 - ft_isprint : Returns "1" if the char is printable

7 - ft_toupper : Turns a lowercase char in a uppercase char

8 - ft_tolower : Turns a uppercase char in a lowercase char
